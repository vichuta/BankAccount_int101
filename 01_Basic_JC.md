# Basic JavaScript Statements
* Semicolon in the end of statement is an optional 
```javascript
    let y=20
    //แนะนำให้ใสเครื่องหมายปิดด้วย
    let x=10; 
```
* Statement can take up multiple lines
* Comment
```
    //Single Line Comment
    /* … */ Single or Multiple Lines Comment
```
* Console Printing
```javascript 
    Console.log (variable);
```
